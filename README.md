# 中国互联网安全合规中心

# Information Hub Certification Center China (IHCCC)

**中国互联网安全合规中心（IHCCC）** 是一个致力于推动互联网安全、信息合规与开放标准建设的非营利性倡议组织。我们通过社区协作的方式，推动网络安全最佳实践、合规基线以及开放安全标准的发展。

**Information Hub Certification Center China (IHCCC)** is a non-profit initiative dedicated to promoting internet security, information compliance, and open cybersecurity standards through community collaboration.

---

## 我们的使命 | Mission

**中文**

* 推动互联网安全与信息合规最佳实践
* 建立开放、透明的安全基线与标准
* 提升组织与个人的网络安全意识
* 促进全球安全社区的协作与共享

**English**

* Promote best practices in internet security and information compliance
* Develop open and transparent security baselines and standards
* Improve cybersecurity awareness across organizations and individuals
* Encourage collaboration within the global security community

---

## 关注领域 | Focus Areas

**中文**

* 互联网基础设施安全
* 主机与服务器安全基线
* 网络安全合规框架
* 开放安全标准与技术文档

**English**

* Internet infrastructure security
* Host and server security baselines
* Security compliance frameworks
* Open cybersecurity standards and technical documentation

---

## 开放标准 | Open Standards

**中文**

IHCCC 致力于开发和发布社区驱动的安全标准、合规基线以及技术规范，为互联网服务、服务器系统和数字基础设施提供参考。

**English**

IHCCC develops community-driven security standards, compliance benchmarks, and technical guidelines to improve the security posture of internet services, servers, and digital infrastructure.
