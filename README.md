# China Internet Security Compliance Center (IHCCC)

**中国互联网安全合规中心（IHCCC）** 是一个致力于推动互联网安全与合规实践的非营利性倡议组织。我们专注于构建开放、透明、社区驱动的网络安全标准、基线和最佳实践。

**China Internet Security Compliance Center (IHCCC)** is a non-profit initiative dedicated to promoting open, transparent, and community-driven practices in internet security and compliance.

---

## 我们的使命 | Mission

**中文**

* 推动互联网安全标准与最佳实践的发展
* 提升组织与个人的安全合规意识
* 构建开放的网络安全标准与基线
* 促进全球安全社区的协作与共享

**English**

* Promote internet cybersecurity standards and best practices
* Improve security and compliance awareness
* Develop open security benchmarks and frameworks
* Encourage collaboration within the global security community

---

## 关注领域 | Focus Areas

**中文**

* 互联网基础设施安全
* 主机与服务器安全基线
* 网络安全合规框架
* 开放网络安全标准与文档

**English**

* Internet infrastructure security
* Host and server security baselines
* Security compliance frameworks
* Open cybersecurity standards and documentation

---

## 开放标准 | Open Standards

**中文**

IHCCC 致力于发布社区驱动的安全标准与合规基线，为互联网服务、服务器系统和数字基础设施提供参考规范。

**English**

IHCCC develops community-driven security standards and compliance benchmarks to help improve the security posture of internet services, servers, and digital infrastructure.
