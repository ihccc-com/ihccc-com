# Internet Host Cybersecurity Compliance Center (IHCCC)

**中国互联网安全合规中心（IHCCC）** 是一个独立的非营利性倡议组织，专注于互联网基础设施安全、主机系统安全基线以及网络安全合规框架的研究与发展。
中心致力于通过开放协作与社区参与，推动网络安全标准、技术规范和最佳实践的制定与传播，以提升互联网服务、服务器系统以及数字基础设施的整体安全水平。

**Internet Host Cybersecurity Compliance Center (IHCCC)** is an independent non-profit initiative dedicated to the development of cybersecurity standards, host security baselines, and compliance frameworks for internet infrastructure.
Through open collaboration and community participation, IHCCC develops security standards, technical guidelines, and best practices aimed at improving the security and compliance posture of servers, systems, and digital services.

---

## 使命 | Mission

**中文**

* 推动互联网基础设施安全与网络安全合规标准的发展
* 建立开放、透明的主机系统安全基线与技术规范
* 推动安全最佳实践在互联网服务与系统环境中的应用
* 促进全球网络安全社区的协作与知识共享

**English**

* Promote the development of cybersecurity standards and compliance practices for internet infrastructure
* Establish open and transparent host security baselines and technical standards
* Encourage the adoption of security best practices across internet services and systems
* Foster collaboration and knowledge sharing within the global cybersecurity community

## 研究与标准领域 | Focus Areas

**中文**

* 互联网基础设施安全（Internet Infrastructure Security）
* 主机与服务器安全基线（Host Security Baselines）
* 网络安全合规框架（Cybersecurity Compliance Frameworks）
* 开放安全标准与技术文档（Open Security Standards and Documentation）

**English**

* Internet infrastructure security
* Host and server security baselines
* Cybersecurity compliance frameworks
* Open security standards and technical documentation

---

## 安全标准 | Security Standards

**中文**

IHCCC 致力于开发社区驱动的网络安全标准与系统加固基线，为服务器系统、互联网服务以及数字基础设施提供参考规范，帮助组织提升整体安全能力与合规水平。

**English**

IHCCC develops community-driven cybersecurity standards and system hardening baselines designed to improve the security and compliance posture of servers, internet services, and digital infrastructure.
